# labali-blog-writing-style

An AI skill that writes and revises personal blog posts in Kingson Wu's established voice — pragmatic, direct, engineering-aware — for Chinese technical essays, workplace reflections, and reading notes.

## What it does

- Drafts new articles from a topic or outline
- Rewrites existing posts to match the author's voice
- Refines AI-generated content to remove abstraction drift
- Improves structure and hierarchy while preserving core meaning

## Supported article types

- Technical summary / deep dive
- Troubleshooting / postmortem
- Workplace reflection
- Reading note / idea note

## Usage

Invoke via Claude Code or a compatible agent:

```
Write a new blog post about [topic] in my established style.
Rewrite this article: [paste article text]
Refine the structure of this draft without changing the core argument.
```
