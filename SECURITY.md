# Security Policy

## Supported Scope

This repository currently contains skill definitions and tooling scripts.
Security issues include:

- Credential leaks in committed files
- Unsafe script behavior that may cause destructive operations
- Supply chain risks from dependency or script changes

## Reporting a Vulnerability

Please do not open public issues for security vulnerabilities.
Instead, use GitHub private vulnerability reporting (if enabled) or contact repository maintainers directly.

Include:

- A clear description of the issue
- Impact and potential risk
- Steps to reproduce
- Suggested mitigation (if available)

We will acknowledge reports as soon as possible and coordinate a fix and disclosure timeline.
